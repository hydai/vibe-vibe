# vibe-vibe

An idea pool for collecting, discussing, and developing ideas.

## What is this?

This repository is an **idea pool** — a place to propose and discuss ideas using GitHub issues. Each idea follows a simple lifecycle:

1. **Submit** — Open a GitHub issue describing your idea.
2. **Discuss** — The community reviews and refines the idea in the issue thread.
3. **Specify** — Accepted ideas get a detailed specification.
4. **Implement** — The specification is turned into code via a pull request.

Want to contribute an idea? [Open an issue](../../issues/new) and describe what you have in mind.

## Managed by Hammurabi

This repository is managed by [Hammurabi](https://github.com/hydai/hammurabi), a tool that automates the issue-to-implementation workflow. Hammurabi watches for new ideas, coordinates the specification process, and drives implementation — turning GitHub issues into merged pull requests.
